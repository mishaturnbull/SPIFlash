# 8-pin flash chip clip pinout

For an 8-pin chip, which will look similar to this:


```
   +------+
---| o    |---
---|      |---
---|      |---
---|      |---
   +------+
```

the pinout will be:

```
               +------+
(white) !CE ---| o    |--- 3v3  (red)
(brown)  SO ---|      |--- !RST
        !WP ---|      |--- SCK  (green)
(black) GND ---|      |--- SI   (blue)
               +------+
```
