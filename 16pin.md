# 8-pin flash chip clip pinout

For an 8-pin chip, which will look similar to this:


```
   +------+
---| o    |---
---|      |---
---|      |---
---|      |---
---|      |---
---|      |---
---|      |---
---|      |---
   +------+
```

the pinout will be:

```
               +------+
            ---| o    |--- CLK  (green)
(red)   3v3 ---|      |--- SI   (blue)
            ---|      |---
            ---|      |---
            ---|      |---
            ---|      |---
(white) !CE ---|      |--- GND  (black)
(brown)  SO ---|      |--- 
               +------+
```
